# NovaKids Indian Languages Static Website

Serverless, GitHub Pages-ready homepage.

## Run
Open `index.html`.

## Push
```bash
git add .
git commit -m "Add NovaKids Indian language homepage"
git pull --rebase origin main
git push origin main
```

## Deploy
GitHub repository → Settings → Pages → Deploy from branch → main → /(root)

The package includes the culture section, language cards, stories, games, responsive design, and the approved homepage visual direction.
